## Rules in editing the code 
- The code needs to be unnecesarily large
  ### Example:
  Instead of:
  ``` lua
  local function q()
    print("Hello")
    playsound()
    tick()
  end

  q()
  ```
  Do this:
  ``` lua
  print("Hello")
  playsound()
  tick()
  ```
- Make it unreadable
  ### Example:
  Instead of:
  ``` lua
  local h = math.random(1,2)
  os.execute("sleep " .. h)
  print(h)
  ```
  Do this:
  ``` lua
  local
  h=math.random(

  1,2)
  os.execute(




  "sleep" ..


  h)
  print(h
  )
  ```
